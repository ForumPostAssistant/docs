How To Use This Template

This test template file provides an alternative Docs Editor option to update & preview specific content sections of the FPA Documentation site without the need to install jekyll locally. It includes all the appropriate styling and javascript libraries as used on the production website and should reasonably accurately simulate the same environment and feel as the production website.

Adding Content To Update...

  • 1) navigate to the _includes/ folder of the Github "docs" repository
  • 2) select/open the relevant content flie from the _includes/ folder that you wish to update
  • 3) copy the COMPLETE content block in the selected file (include any html and text)
  • 4) paste the COMPLETE content block between the TEST SECTION comment tags in the source of this file

Commiting Updated Content...

  • 1) open an issue on the Github repository, assigned to yourself and tagged as content update for team review
  • 2) copy the COMPLETE updated content block from this test file source (include any html and text)
  • 3) paste the COMPLETE updated content block between the TEST SECTION comment tags in to the new "content update" issue
